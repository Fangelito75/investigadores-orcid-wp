# Investigadores + ORCID

**Investigadores + ORCID** is a WordPress plugin for managing research staff profiles with automatic retrieval and display of scientific publications from ORCID.

## Features
- Management of research staff profiles
- ORCID integration (public API)
- Automatic import of publications
- Customizable design (colors, icons, buttons)
- Institutional-ready frontend templates
