== Description ==
- Fichas de investigadores con imagen destacada (foto), datos básicos y enlaces (ORCID, LinkedIn, Google Scholar, CV).
- Importación de publicaciones públicas desde ORCID y almacenamiento en un CPT para mostrarlas en la ficha.
- Shortcodes: [investigadores] y ficha individual mediante template propio.
- Diseño personalizable mediante variables CSS desde Ajustes.

== Installation ==
1. Subir el plugin y activarlo.
2. Crear investigadores en el menú "Investigación".
3. Insertar el shortcode [investigadores] en una página para mostrar el directorio.
